# 🥗 Cardápio de Saladas

Este projeto é uma página simples em HTML que apresenta um **cardápio semanal de saladas**, com opções variadas e ingredientes detalhados para cada dia da semana.

## 📌 Sobre o projeto

O objetivo deste projeto é praticar a estrutura básica do HTML, utilizando elementos como:

* Tabelas (`<table>`)
* Títulos (`<h1>`, `<h3>`)
* Formatação de texto (`<strong>`, `<i>`, `<em>`, `<small>`)
* Inserção de imagens (`<img>`)
* Organização de conteúdo

A página simula um cardápio saudável, com foco em alimentação equilibrada e nutritiva.

## 📅 Cardápio semanal

| Dia     | Salada           |
| ------- | ---------------- |
| Domingo | Salada Caesar    |
| Segunda | Salada Grega     |
| Terça   | Salada Caprese   |
| Quarta  | Salada de Quinoa |
| Quinta  | Salada de Frango |
| Sexta   | Salada de Atum   |
| Sábado  | X-Salada         |

Cada salada contém seus ingredientes descritos diretamente na página.
